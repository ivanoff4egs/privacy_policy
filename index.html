Политика конфиденциальности

Это тестовое приложение использует Instagram Graph API для отправки и получения сообщений.

Данные пользователей, полученные через API, не сохраняются и не передаются третьим лицам.

Это приложение используется исключительно для тестирования и разработки функционала сообщений в Instagram.

Если у вас есть вопросы по поводу конфиденциальности, пожалуйста, свяжитесь с разработчиком приложения.
